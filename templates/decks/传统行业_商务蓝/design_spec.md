# 传统行业_商务蓝 - 视觉设计规范

## I. 模板概述

| 属性 | 描述 |
| --- | --- |
| **模板名称** | 传统行业_商务蓝 |
| **展示名称** | 传统行业经典深蓝商务汇报模板 |
| **适用场景** | 适用于传统行业（商务汇报、装备制造、基建工程汇报与战略分析等）的汇报演示。 |
| **设计基调** | 专业、沉稳、精炼、工业感 |
| **主题模式** | 蓝灰修饰的白底风格内容页 + 深蓝底色封面/结束页 |

---

## II. 画布规格

| 属性 | 数值 |
| --- | --- |
| **比例** | 标准 16:9 |
| **分辨率** | 1280 × 720 px |
| **viewBox** | `0 0 1280 720` |
| **安全边距** | 左右 56px, 上 48px, 下 40px |

---

## III. 色彩规范

| 角色 | 颜色值 | 用途 |
| --- | --- | --- |
| **主色（深蓝）** | `#485275` | 顶部/底部修饰角、文字标题、大底色背景 |
| **浅灰色** | `#F2F2F2` | 目录框和内容分割线的背景填充色 |
| **背景灰色** | `#F6F6F6` | 画布的大背景底色 |
| **白字/浅色** | `#FFFFFF` | 深蓝底色上的主文本 |
| **深黑字** | `#404040` | 浅色底色上的主体正文 |

## V. 页面列表 (Page Roster)

| 文件 | 角色 | 描述 |
| ------ | ------ | ------------- |
| `01_cover.svg` | cover | 封面页；包含主标题、副标题、以及背景几何造型 |
| `02_toc.svg` | toc | 目录页；包含四个章节索引条目与大号数字标识 |
| `02_chapter.svg` | chapter | 章节页；大序号、章节标题及章节描述文本 |
| `03_content.svg` | content | 内容页；包含顶部大标题、辅助描述以及三个带菱形裁剪插图的条目 |
| `04_ending.svg` | ending | 结束页；庄重的收尾排版，呼应封面经典深蓝 |
