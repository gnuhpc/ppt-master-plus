# 传统行业_经典红 - 视觉设计规范

## I. 模板概述

| 属性 | 描述 |
| --- | --- |
| **模板名称** | 传统行业_经典红 |
| **展示名称** | 传统行业经典党政与国企汇报红金色模板 |
| **适用场景** | 适用于传统行业（党政建设、国企汇报、大型工业制造战略分析等）的汇报演示。 |
| **设计基调** | 庄重、大气、严谨、金色与红色经典搭配 |
| **主题模式** | 红色喜庆背景 + 金黄文字强调 |

---

## II. 画布规格

| 属性 | 数值 |
| --- | --- |
| **比例** | 标准 16:9 |
| **分辨率** | 1280 × 720 px (缩放自 960 × 540) |
| **viewBox** | `0 0 1280 720` |
| **安全边距** | 左右 56px, 上 48px, 下 40px |

---

## III. 色彩规范

| 角色 | 颜色值 | 用途 |
| --- | --- | --- |
| **主色（党政红）** | `#CD170A` | 背景底色，版面骨架，红色基调 |
| **辅色（经典金黄）** | `#FFEBA9` | 重点标题渐变 stop-color 之一，圆形高亮边框 |
| **次色（金黄渐变深色）** | `#E29B60` | 金色文字渐变 stop-color 之一 |
| **浅黄文字** | `#F6E0CB` | 辅助段落文本，页眉辅助修饰文本 |
| **白字/浅字** | `#FFFFFF` | 主段落正文文本 |

---

## IV. 字体系统

| 级数 | 用途 | 字号 | 字形 |
| --- | --- | --- | --- |
| **H1** | 封面主标题 | 88px | 粗体 / 汉仪大宋简 |
| **H2** | 章节/大标题 | 58px | 粗体 / 汉仪大宋简 |
| **H3** | 内容页标题 | 26px | 粗体 / 汉仪大宋简 |
| **Body** | 正文文本 | 16px | 常规 / Century Gothic |
| **Caption** | 页脚辅助信息 | 14px | 常规 / Century Gothic |

## V. 页面列表 (Page Roster)

| 文件 | 角色 | 描述 |
| ------ | ------ | ------------- |

| `02_toc.svg` | toc | 目录页；包含四个带有金色序号的章节索引条目 |

| `04_ending.svg` | ending | 结束页；庄重的收尾排版，呼应封面经典红底与金字 |
