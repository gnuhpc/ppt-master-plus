本指南专注于您在制作或美化 PPT 时的**具体操作动作和交互步骤**。通过明确的动作指令，您可以最高效地控制制作流程并获得高品质的可编辑幻灯片。

## 1. 第一步：明确制作意图与输入准备

在向 AI Agent 发送第一条指令前，您需要决定是“重构故事”还是“排版美化”。

### 动作 A：重构故事（从文档新建 PPT）
如果您提供的是 PDF、Word、Excel、网页链接或一段乱序文本，且**允许 AI 自由调整页面顺序、合并或拆分内容、改写文字**：
*   **指令动作**：在聊天框中输入您的源文件路径/链接，并明确指定受众、目标页数及**逐页精修**模式。
*   **示例指令**：
    > 帮我把 `/path/to/product_spec.pdf` 重新组织，制作成一份 10 页左右的商业路演 PPT。目标受众是投资人，风格要求专业冷色系，走逐页确定精修模式。

### 动作 B：排版美化（美化已有 PPT）
如果您提供的是一份现有的 PPTX，且**必须保证原幻灯片页数、页面顺序、每页的文字和数据完全不改变**，只优化排版、配色、字号与图表：
*   **指令动作**：在输入已有 PPTX 文件的同时，必须包含“美化”、“保留文字不变”、“页数不变”等限定词，触发 `beautify` 流程。
*   **示例指令**：
    > 使用 `ppt-master-plus` 美化 `/path/to/draft.pptx`，保留每页的文字内容和页面顺序完全不变，只重新设计版式和留白，使其更加商务专业。
*   *注意：请勿尝试将已有的 PPTX 作为填充新内容的“模板”。如果需要套用已有 PPTX 的样式来做新内容，请走动作 A（重构故事）并以该 PPTX 为参考事实。*

## 2. 第二步：在网页端确认全局设计规范 (Confirm UI)

在 AI 导入并解析完源内容后，将自动为您启动一个本地交互网页服务。

### 核心操作动作
1.  **打开确认页面**：在 Agent 给出如下提示时，**直接点击**终端或对话中输出的 URL（默认通常是 `http://localhost:5050`）：
    ```text
    已启动设计确认服务，请在浏览器中打开：http://localhost:5050 进行确认。
    ```
2.  **Tier 1（方向确认）**：
    *   在页面左侧核对画布比例（如 `16:9`）、受众与核心大纲。
    *   在 **"Delivery Purpose / 交付目的"** 中进行单选：
        *   若需要字多、高密度的报告，选择 **Text (20px)**；
        *   若需要平衡版式，选择 **Balanced (24px)**；
        *   若需要演讲大字，选择 **Presentation (32px)**。
    *   在 **"Visual Style"** 下拉框中，选择您的视觉风格基调。
    *   点击页面底部的 **"Next / 下一步"**。
3.  **Tier 2（细节确认）**：
    *   **选择配色**：页面会呈现 3 组根据 Tier 1 自动推导出的色卡。**直接点击您中意的色卡进行勾选**。如需使用公司品牌色，可在 "Custom" 输入框内直接输入十六进制 HEX 颜色（如 `#003399`）。
    *   **选择字体**：点击选择系统推荐的 CJK/拉丁字体组合。
    *   **指定图片策略**：若不需要图片，勾选 `none`；若允许 AI 生成，勾选 `ai`；若需要您自己后续放入，勾选 `provided`。
    *   **完成确认**：点击页面右下角的 **"Confirm / 确认"** 按钮。网页会自动保存配置并关闭服务。

### 网页打不开时的备用动作
如果您处于无浏览器环境（如远程服务器），请在 Agent 对话框中直接回复：
> 网页无法打开，请直接在聊天中向我展示 Tier 1 确认项。
> *(之后根据 Agent 列出的选项，在聊天中直接回复序号或参数值)*

## 3. 第三步：利用浏览器进行实时预览与交互修改 (Live Preview)

当 PPT 开始一页页生成时，预览网页（`http://localhost:5050`）会实时同步显示已生成幻灯片的 SVG 效果。

### 动作 A：快速修改微调（无需 Agent 重新生成）
如果您只想改个错别字、调个颜色或移动一下某个元素的位置：
1.  在网页预览中，**直接鼠标点击**需要修改的文字、图标或形状。
2.  在右侧弹出的属性面板中，直接输入新的文字，或调整 `x`/`y`/`width`/`height` 数值，或选择新的填充颜色。
3.  点击右侧面板底部的 **"Apply changes / 应用修改"**，您的改动将立即写入项目文件。

### 动作 B：描述布局重构（需要 Agent 重新设计）
如果某页的排版不符合预期，需要重新排版或调整结构：
1.  在右侧的 **"Annotations / 批注"** 文本框中，用自然语言写下您的具体修改要求。
    *   *示例*：“把中间的 4 个卡片改用 2x2 的网格排列，背景改用浅灰色，突出第一个卡片。”
2.  点击 **"Add annotation / 添加批注"**。
3.  点击 **"Apply changes / 应用修改"**。
4.  此时页面会弹出一个包含批注信息的命令块，**点击复制该提示词**。
5.  回到 Agent 的聊天窗口，**直接粘贴该提示词**并发送（或者直接对 Agent 说：“应用我刚才在浏览器里添加的批注并重新生成”）。

## 4. 第四步：管理与更新图片/公式资源

PPT 中使用的所有图片和数学公式均作为独立资源由您或 AI 共同管理。

### 动作 A：使用您自己的本地配图
1.  在制作开始前或暂停时，**直接将图片文件（`.png` / `.jpg`）复制**到您项目目录下的 `images/` 文件夹中。
2.  在设计规范确认阶段，将该图片的 `Acquire Via` 设置为 `user`，并在 `Filename` 列中填写对应的文件名。

### 动作 B：手动精修/生成 AI 图片
如果您在设计阶段选择了 `ai` 图片策略，但对 AI Agent 自动生成的图片不满意：
1.  在您的项目目录下，打开 `images/image_prompts.md` 文件。
2.  该文件中已自动列出了每一张幻灯片配图的详细中英文 AI 生成提示词（Prompts）。
3.  **复制提示词**，直接粘贴到您自己的 Midjourney 或 DALL-E 工具中进行生成。
4.  将下载的满意图，重命名为文件中要求的占位文件名（例如 `hero_banner.png`），覆盖放回 `images/` 文件夹中。

### 动作 C：公式强制渲染
1.  如果在文档中包含复杂的 LaTeX 数学公式，且您希望在任意电脑上打开 PPT 均不会因缺少字体而出现乱码：
2.  在全局确认（Typography）阶段，将公式策略选择为 `mixed` 或 `render-all`。
3.  Agent 将在后台自动将公式编译为透明的高清 PNG 图片并精准排版，您无需任何手动导出操作。

## 5. 第五步：导出幻灯片与追加动画/配音

在所有单页制作并预览通过后，即可进行最终的导出与效果增强。

### 动作 A：执行导出
*   **动作**：在聊天窗口向 Agent 发送导出指令。
*   **示例指令**：
    > 所有页面预览都没问题了，请导出最终的 PPTX。
*   **动作**：前往您项目目录下的 `exports/` 文件夹，获取生成的 `.pptx` 文件。

### 动作 B：添加动态转场与元素动画
默认导出的幻灯片是静态无动画的。如果您需要为演示文稿增加动效：
*   **整体转场动作**：告诉 Agent 您需要的切换效果。
    *   *示例*：“重新导出 PPTX，添加淡入淡出（`fade`）转场动画。”
*   **单页元素飞入动作**：如果您希望页面上的文字和卡片根据点击依次出现：
    *   *示例*：“为导出的 PPTX 开启元素淡入动画，并设置为点击后触发（`on-click`）。”

### 动作 C：合成语音配音与自动播放
如果您希望将 PPT 制作成自动讲解播放的演示视频，或需要自带旁白音轨：
1.  **动作**：向 Agent 申请生成配音。
    *   *示例*：“请根据 speaker notes 为每一页生成配音，并嵌入到 PPTX 中。”
2.  **动作**：在 Agent 提供的配音演员（TTS 声音）列表中选择一个角色并确认。
3.  合成完成后，打开导出的 PPTX，幻灯片已自动嵌入对应音频，且每页的自动切换时间已与语音时长精准对齐。
